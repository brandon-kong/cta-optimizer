# CTA Optimal Route Finder

Apps such as Google Maps and Ventra provide users with the ability to find the fastest routes from one location to another. However, these apps don't consider cost, security, or other factors that may be important to the user. This project aims to provide a more customizable route-finding experience for users of the Chicago Transit Authority (CTA) system. The user can input their preferences, such as cost, security, and time, and the program will output the optimal route based on these preferences.
